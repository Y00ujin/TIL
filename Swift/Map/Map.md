### Map이 뭐야?

> #### ```스위프트의 대표적인 매개변수로 함수를 갖는 함수```
> #### ```컨테이너가 담고 있던 각각의 값을 매개변수를 통해 받은 함수에 적용한 후 새로운 컨테이너를 생성하여 반환 (기존 컨테이너의 값은 변경되지 않음)```

- #### 사용해보자
```
    let array = [0, 1, 2, 3]
    let newArray = array.map { $0 * 5 }
    print(newArray) //[0, 5, 10, 15]
```
